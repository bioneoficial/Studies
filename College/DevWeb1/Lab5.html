1) Qual o objetivo das tags abaixo?
style
Tag onde estilizamos o css
font family
Define a familia da fonte
font size
Define o tamanho da fonte
background-color
Define a cor de fundo
background-image
Define uma imagem de fundo
background-repeat
Define se o fundo repete, ou nao, alinhamento em eixo x ou y,etc.
background-size
Define o tamanho do fundo
background-attachment
Define se a imagem é fixa ou se scrolla com o resto da page
background-position
define a posicao do fundo, esquerda direita centro etc
font-weight
Define propriedade da fonte como Bold, mais grosso ou mais fino as letras.
margin
A margem entre o elemento e o resto da page, outro elemento por exemplo, é externo.
padding
a margem entre o elemento e o conteudo, é interno.
float
define uma posicao float pro elemento, right left.
display
Define um comportamento pro elemento (inline, block, flex etc)
text-shadow
Sombra no texto
box-shadow
Sombra de caixa, sombra que vem por tras do conteudo estilizado
cursor
Muda o cursor(simbolo do mouse)
text-align
Alinhamento horizontal do texto
opacity
Define a opacidade
font-weight
Define propriedade da fonte como Bold, mais grosso ou mais fino as letras.
hover
Pseudo que é usado quando o mouse passa por cima do elemento.
border-radius
Define as pontas da borda
